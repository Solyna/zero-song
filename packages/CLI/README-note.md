## commander —— 命令行指令配置

第三方库 commander 来实现脚手架命令的配置。

- Step1: 安装 commander 依赖，并导入 demo 项目中: `npm install commander`

## chalk —— 命令行美化工具

安装 chalk 时一定要注意安装 4.x 版本(小包使用的是 4.0.0)，否则会因为版本过高，爆出错误。
`npm install chalk@4.0.0`
